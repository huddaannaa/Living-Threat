Help & Guide Living MITRE Repository

What this platform does

A living threat repository: real-world incidents and write-ups are continuously mapped to ATT&CK tactics and techniques. The UI is designed for analyst flow — report → techniques → hunt/detect → pivots.

Core features

Navigator view
Techniques are grouped by tactic. Clicking a technique opens its evidence context. Selecting a document enables Hunt Mode and Attack Path highlighting.
Filters
Range (7/30/90/180/365/All), Minimum severity, doc_summary search, and Actor/Tool pivots. Filters affect the documents list and the matrix.
Attack Path + Hunt Pack
Shows kill-chain phases and highlights phases that actually contain techniques. Hunt Pack includes runbook, checks, pivots, coverage/gaps, exfil watch, and integrity.
Fast navigation
Ctrl+K Quick Switcher, desktop tabs, and mobile docs drawer.

Updates / performance

Refresh loads the latest dataset (BOOTSTRAP_SIZE). “New docs” badge uses a light heartbeat poll. No automatic full refresh loop — avoids reloading thousands of docs repeatedly.

About Open Source

The project

Living MITRE Repository is built for practical threat work: stable technique IDs, evidence you can review quickly, and mappings that stay readable when you’re moving fast.
